# Slack notifications (Beta)

Boxty can send notifications to Slack channels.

## Prerequisites

- A Slack workspace
- Admin access to the Slack workspace
- A Boxty workspace on the Team or Enterprise plan

## Configuration

1. Go to Settings > Integrations in the Boxty dashboard
2. Click "Add Integration" and select Slack
3. Authorize Boxty to access your Slack workspace
4. Select the channels to notify
5. Choose which events to notify on (deployments, failures, etc.)
6. Click "Save"

Boxty will now send notifications to the configured Slack channels.
