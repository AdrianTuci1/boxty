# GPU Metrics

Boxty provides detailed GPU metrics for monitoring and optimization.

## GPU utilization %

GPU utilization measures the percentage of time the GPU is actively processing. Low utilization may indicate:

- CPU bottleneck
- Memory bandwidth limitation
- Inefficient kernel code
- Too small batch size

Monitor GPU utilization in the Boxty dashboard and adjust your workload accordingly.

Other available metrics include:

- Memory utilization
- Tensor core utilization
- PCIe bandwidth
- Temperature
- Power draw
