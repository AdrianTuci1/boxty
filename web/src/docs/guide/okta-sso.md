# Okta SSO

Boxty supports Okta SSO for enterprise authentication.

## Prerequisites

- An Okta account with admin access
- A Boxty workspace on the Enterprise plan

## Configuration

1. Go to Settings > SSO in the Boxty dashboard
2. Select "Okta" as the provider
3. Enter your Okta domain and API token
4. Configure attribute mappings
5. Click "Save"

Your team members can now log in to Boxty using their Okta credentials.
