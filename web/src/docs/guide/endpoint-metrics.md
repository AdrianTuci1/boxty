# Endpoint metrics

Boxty provides detailed metrics for monitoring your endpoints.

## What the metrics mean

| Metric | Description |
|--------|-------------|
| Requests/sec | Number of requests processed per second |
| Latency (p50) | Median response time |
| Latency (p99) | 99th percentile response time |
| Error rate | Percentage of failed requests |
| GPU utilization | Percentage of GPU capacity used |
| Memory usage | Current memory consumption |
| Cold starts | Number of container cold starts |

These metrics are available in the Boxty dashboard and via the API.
