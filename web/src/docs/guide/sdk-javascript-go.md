# JavaScript/Go SDKs

Boxty provides SDKs for JavaScript and Go in addition to Python.

## Installation

### JavaScript

```bash
npm install @boxty/sdk
```

### Go

```bash
go get github.com/boxty/sdk-go
```

These SDKs allow you to invoke Boxty Functions and manage resources from JavaScript and Go applications.

Note: Function definitions still require Python. The JavaScript and Go SDKs are for client-side interaction.
