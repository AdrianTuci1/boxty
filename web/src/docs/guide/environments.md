# Environments

Boxty Environments let you organize your Apps and resources into isolated groups.

## Create a Workspace

Workspaces are the top-level organizational unit in Boxty. Each workspace has its own set of environments, users, and billing.

To create a workspace:

1. Go to the Boxty dashboard
2. Click "Create Workspace"
3. Enter a name and description
4. Select a plan
5. Click "Create"

You can then create environments within your workspace for development, staging, and production.
