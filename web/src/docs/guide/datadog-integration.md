# Connecting Boxty to your Datadog account

This guide walks you through integrating Boxty with Datadog for monitoring and observability.

## What this integration does

The Datadog integration sends Boxty metrics and logs to your Datadog account, allowing you to monitor your Boxty applications alongside your other infrastructure.

## Installing the integration

1. Go to Settings > Integrations in the Boxty dashboard
2. Click "Add Integration" and select Datadog
3. Enter your Datadog API key and site URL
4. Select which metrics and logs to forward
5. Click "Save"

## Metrics

The following metrics are sent to Datadog:

- Function execution count
- Function execution duration
- Container startup time
- GPU utilization
- Memory usage
- Disk usage
