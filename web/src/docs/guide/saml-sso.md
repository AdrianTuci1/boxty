# Custom SAML SSO

Boxty supports custom SAML SSO providers for enterprise authentication.

## Prerequisites

- A SAML 2.0 identity provider
- A Boxty workspace on the Enterprise plan

## Configuration

1. Go to Settings > SSO in the Boxty dashboard
2. Select "Custom SAML" as the provider
3. Enter your IdP metadata URL or XML
4. Configure attribute mappings
5. Click "Save"

Your team members can now log in to Boxty using your SAML identity provider.
