# Role-Based Access Control (RBAC)

Boxty provides RBAC for managing access to your workspace.

## Workspace Roles

| Role | Permissions |
|------|------------|
| Owner | Full access to all resources |
| Admin | Manage users, billing, and settings |
| Developer | Deploy apps, manage functions |
| Viewer | Read-only access |

## Environment Roles

You can also assign roles per environment:

| Role | Permissions |
|------|------------|
| Environment Admin | Full access to the environment |
| Environment Developer | Deploy and manage apps in the environment |
| Environment Viewer | Read-only access to the environment |

Configure roles in Settings > Members.
