# VM Sandboxes (Alpha)

Boxty VM Sandboxes provide full virtual machine isolation.

## Improvements over gVisor sandboxes

VM Sandboxes offer:

- Full OS isolation
- Custom kernel support
- Persistent disk
- Direct hardware access
- Better performance for some workloads

VM Sandboxes are in Alpha and available on the Enterprise plan. Contact sales@boxty.com for access.
