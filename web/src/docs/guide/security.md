# Security and privacy

Boxty is designed with security and privacy at its core.

## Application security (AppSec)

Boxty implements multiple layers of application security:

- **Sandboxing**: Untrusted code runs in isolated sandboxes
- **Network isolation**: Functions run in private networks by default
- **Encryption**: All data is encrypted at rest and in transit
- **Access control**: RBAC for fine-grained permissions
- **Audit logging**: Track all sensitive actions

## Corporate security (CorpSec)

Boxty's corporate security program includes:

- SOC 2 Type II compliance
- Regular security audits
- Penetration testing
- Bug bounty program
- Employee security training

Contact security@boxty.com for security inquiries or to report vulnerabilities.
