# Workspaces

Boxty Workspaces are the top-level organizational unit for your team.

## Create a Workspace

Create a workspace from the Boxty dashboard:

1. Go to the Boxty dashboard
2. Click "Create Workspace"
3. Enter a name and description
4. Select a plan (Starter, Team, or Enterprise)
5. Click "Create"

Each workspace has:

- Independent billing
- User management
- Environment configuration
- Resource quotas
- Audit logs

Invite team members to your workspace and assign roles based on their responsibilities.
