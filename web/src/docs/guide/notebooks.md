# Modal Notebooks

Boxty Notebooks provide a managed Jupyter environment for interactive development.

## Getting started

Create a new notebook from the Boxty dashboard:

1. Go to Notebooks in the dashboard
2. Click "Create Notebook"
3. Select a kernel configuration (CPU, GPU, etc.)
4. Start coding

## Kernel resources

Configure your notebook kernel with the resources you need:

- CPU cores
- Memory
- GPU type
- Disk space

Boxty automatically provisions and scales resources based on your configuration.
