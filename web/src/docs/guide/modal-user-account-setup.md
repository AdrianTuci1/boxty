# Boxty user account setup

This guide walks you through setting up your Boxty user account.

## What GitHub permissions does signing up require?

Boxty uses GitHub OAuth for authentication. When you sign up, Boxty requests:

- Read access to your public profile
- Read access to your email address

Boxty does not request write access to your repositories or any other sensitive permissions.

After signing up, you can create workspaces, invite team members, and start deploying applications.
