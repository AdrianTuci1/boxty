TASK: Implementează Node.js SDK-ul Boxty (TypeScript).

Citește `/tmp/repo/AGENT_SDK_JS.md` pentru specificația completă. Directorul tău de lucru: `/tmp/repo/sdk-js/`.

**package.json** — ES modules, TypeScript, bin entry `boxty` (compiled JS), dependențe: ws, commander, node-fetch (sau use native fetch)

**tsconfig.json** — strict mode, ES2022, moduleResolution node

**src/index.ts** — exportă: Client, Sandbox, Image, App, Workspace, Environment, Deployment, Schedule, Secret + types

**src/types.ts** — toate interfețele:
- ClientOptions, SandboxOptions, ExecResult
- SandboxMetrics, AppMetrics, AppOptions
- ScheduleOptions, Schedule interface
- Workspace, Environment, Secret, Deployment interfețe
- Volume, ImageBuild interfețe

**src/client.ts** — Client class:
- createSandbox, listSandboxes, getSandbox, deleteSandbox, getSandboxMetrics
- buildImage, listImages, deleteImage
- balance, usage, buyCredits
- createSecret, listSecrets, deleteSecret, attachSecrets
- createSchedule, listSchedules, deleteSchedule, triggerSchedule
- createVolume, listVolumes, deleteVolume, mountVolume, unmountVolume
- createWorkspace, listWorkspaces, getWorkspace, deleteWorkspace
- createEnvironment, listEnvironments
- createApp, listApps, getApp, deleteApp, stopApp, deployApp, getAppSandboxes, getAppDeployments, getAppMetrics, getAppUsage, getAppLogs

**src/sandbox.ts** — Sandbox class (extends EventEmitter): exec, forward, checkpoint, stop, metrics, attachSecrets, on('stdout')/on('stderr')/on('exit'), static restore()

**src/errors.ts** — BoxtyError, AuthenticationError, NotFoundError, etc.

**src/cli/index.ts** — CLI cu commander:
- run, deploy, shell, exec, ls, logs, stop, cp, forward
- login, whoami, billing
- secret create/ls/rm, schedule create/ls/rm/trigger
- image build/ls/rm, workspace create/ls/rm, app create/ls/stop/logs/metrics

JSDoc pe toate exporturile. examples/, test/
