- `sandboxes.ts` — CRUD + metrics

- `schedules.ts` — CRUD + trigger

- `useSandboxes.ts`, `useApps.ts`, `useWorkspaces.ts` — react-query hooks

**Components (/tmp/repo/web/src/components/):**
- `Layout.tsx` — sidebar + top bar + breadcrumbs

9. `SecretsPage.tsx` — listă, create modal (value masked), delete
10. `ImagesPage.tsx` — listă, build modal
